Electron desktop app serving as an opinionated, terminal integrated session manager for SWE coding agents, giving you a unified UI to run tasks, manage git worktrees/branches, and review diffs across multiple AI agent sessions and projects.

## Commands

- `npm install` — install dependencies
- `npm run dev` — start Electron in dev mode
- `npm run build` — build production Electron app
- `npm run typecheck` — TypeScript check
- `npm test` — run vitest
